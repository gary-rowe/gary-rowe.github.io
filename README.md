# Procedures

## New article

1. Copy most recent article in `_posts` as a baseline
1. Edit as required
1. View on localhost using `bundle exec jekyll serve --watch`
1. When ready, git commit and push to `master`
1. Verify on [https://gary-rowe.com](https://gary-rowe.com)
